# Childhood Chapter Milestones — Deferred Narrative Note

This note parks the current direction for the childhood chapter so technical work can return to Vertical Slice A.

The childhood chapter should not simulate all 18 years linearly. It should use selected milestone years that clearly show changes in the protagonist's awareness, personality, ideals, and dreams.

Suggested milestone structure:

- **Age 8 — The world feels whole**: hometown as the protagonist's first universe; early friendship, family warmth, school, chores, wonder, and the first starry-sky memory.
- **Age 11-12 — First cracks**: the protagonist first notices money, class, adult limitations, loss, or broken promises.
- **Age 14-15 — Self separates from home**: adolescence, personal dreams, conflict with family expectations, friendship strain, and the first clear sense of wanting a life beyond the village.
- **Age 17-18 — Threshold**: the end of Act 1, where a choice, opportunity, or crisis makes leaving home necessary.

Rough pacing target: 15-20 selected gameplay days, not continuous years. Each age band should contain only significant days that reveal a new layer of the protagonist compared with prior milestones.

Core wound candidate to revisit later: the protagonist feels valuable only when becoming the person family or hometown expectations require. The city journey becomes the first attempt to live as a separate self.
